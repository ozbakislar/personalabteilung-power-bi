# 👥 Personalanalyse-Dashboard

Dieses Projekt ist ein interaktives Power BI Dashboard, das Personaldaten eines fiktiven Unternehmens visualisiert und analysiert. Ziel war es, aussagekräftige Einblicke in drei zentrale Bereiche zu gewinnen: die Mitarbeiterverteilung nach Abteilung und Beschäftigungsstatus, das demografische Profil der Belegschaft sowie die wichtigsten Kennzahlen wie Gesamtgehalt, Mitarbeiteranzahl, Frauenanteil und Fluktuationsrate. Das Dashboard wurde mit einem einheitlichen blauen Farbschema gestaltet und ermöglicht durch interaktive Filter (Geschlecht, Status, Stadt) eine intuitive Bedienung.
